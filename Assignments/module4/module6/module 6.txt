1. Explain the structure and purpose of forms in Flutter.
ANS.
In Flutter, forms are created using the Form widget.
It contains multiple TextFormField widgets.
A GlobalKey<FormState> is used to manage and validate the form.

Purpose:
Collect user input (like login, registration, feedback).
Validate data before submission.
Manage form state easily.

2. Describe how controllers and listeners are used to manage form input.
ANS.
->TextEditingController:Used to read and control text field values.
Example: Get email or password value.

->Listeners:Used to detect changes in text fields.
Example: Show live validation or character count.

->Controller helps to get/set text.
->Listener helps to track changes.
